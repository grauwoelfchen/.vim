# .vim

Personal `.vimrc`, color schema, scripts and settings.


## Repository

https://gitlab.com/grauwoelfchen/DotVim


## Setup

```zsh
% git clone git@gitlab.com:grauwoelfchen/dotvimperator.git
% cd .vim
% mkdir .{backup,swap}
% git submodule update --init
% vim
:NeoBundleInstall
```
