# .vim

Personal .vimrc, color schema, script and settings.

```
# lines count is kept < 500
% wc -l .vimrc
483 .vimrc
```

## Setup

```
% git clone git@github.com:grauwoelfchen/.vim.git
% cd .vim
% mkdir .{backup,swap}
% git submodule update --init
% vim
:NeoBundleInstall
```
