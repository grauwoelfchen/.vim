# .vim

Personal .vimrc, color schema and settings

## Setup

```
% git clone git@github.com:grauwoelfchen/.vim.git
% cd .vim
% mkdir .{backup,swap}
% git submodule update --init
% vim
:NeoBundleInstall
```
